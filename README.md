# DebtTracker

DebtTracker 是一个**用于记录欠款、还款与余额变动的轻量级债务追踪系统**，适合个人、家庭、小团队或小型企业使用，目标是让“谁欠谁、欠多少、还了多少、现在还剩多少”**始终清晰、可追溯、可核对**。



## 核心功能

- 记录欠款（借出 / 借入）
- 记录多次还款（部分还款、分期还款）
- 自动计算当前未结清余额
- 支持备注、时间、对方信息的完整留痕
- 所有变动可追溯，不覆盖历史记录
- 适合后续扩展为 Web / App / ERP 子模块



## 设计理念

- **以“流水”为核心，而不是只存一个余额**
- 所有金额变动都有来源、有时间、有说明
- 允许真实世界中的不规则还款方式
- 不做过度金融假设（不强制利息、不强制周期）
- 数据结构清晰，便于审计和扩展



## 基本概念

- **Debt（债务）**
   一次欠款关系的建立，例如：A 借给 B 1000 元。
- **Repayment（还款）**
   针对某一笔债务的还款记录，可以有多次。
- **Balance（余额）**
   = 欠款总额 − 已还款总额（由系统计算，不直接存）



## 示例使用场景

- 个人：
  - 朋友借钱、多次还款，防止记混
- 家庭：
  - 家庭成员之间的垫付与结算
- 小团队 / 小公司：
  - 内部垫款、员工暂借、客户欠款跟踪
- 可作为 ERP / 财务系统中的 **Debt / AR / AP 子模块**



## 快速开始

- 安装依赖：`npm install`
- 开发启动：`npm run dev`
- 打包 Web 版本：`npm run build`



## Windows 桌面版（免安装可执行文件）

项目内置了一个最小化的 Electron 包装层，可以在保留现有界面和功能的前提下，生成免安装的 Windows 可执行文件。

1. 安装依赖（首次执行）：`npm install`
2. 构建前端资源：`npm run build`
3. 打包 Windows 便携版：`npm run electron:build:win`

生成的便携版 `.exe` 位于 `release/` 目录，可直接运行，无需安装。若要产出 Windows 可执行文件，请在 Windows 环境或支持 Windows 目标的交叉编译环境中运行上述命令。
