# DebtTracker

DebtTracker 是一个**用于记录欠款、还款与余额变动的轻量级债务追踪系统**，适合个人、家庭、小团队或小型企业使用，目标是让“谁欠谁、欠多少、还了多少、现在还剩多少”**始终清晰、可追溯、可核对**。



## 核心功能

- 记录欠款（借出 / 借入）
- 记录多次还款（部分还款、分期还款）
- 自动计算当前未结清余额
- 支持备注、时间、对方信息的完整留痕
- 所有变动可追溯，不覆盖历史记录
- 适合后续扩展为 Web / App / ERP 子模块



## 设计理念

- **以“流水”为核心，而不是只存一个余额**
- 所有金额变动都有来源、有时间、有说明
- 允许真实世界中的不规则还款方式
- 不做过度金融假设（不强制利息、不强制周期）
- 数据结构清晰，便于审计和扩展



## 基本概念

- **Debt（债务）**
   一次欠款关系的建立，例如：A 借给 B 1000 元。
- **Repayment（还款）**
   针对某一笔债务的还款记录，可以有多次。
- **Balance（余额）**
   = 欠款总额 − 已还款总额（由系统计算，不直接存）



## 示例使用场景

- 个人：
  - 朋友借钱、多次还款，防止记混
- 家庭：
  - 家庭成员之间的垫付与结算
- 小团队 / 小公司：
  - 内部垫款、员工暂借、客户欠款跟踪
- 可作为 ERP / 财务系统中的 **Debt / AR / AP 子模块**
